I was using the backend code modification to create blogs. But then, I thought it might be a better idea to create an editor that can do it for me from the website itself. A rich text one. And which can take images. And links. And boy, what a rabbit hole it turned out to be. Used the Oauth from Cloudflare into Github as these posts update on github and then get deployed through Vercel to the world. Integration was pain, even while using AI to do it. The more I think about, the more I realize that most of the tech on software stands on critical small legs. Anything can fail. Screwing everyone. Anytime. The whole tech somehow balances itself and the world somehow turns. Everyday.
